Introduction to String Literals
Definition: String literals are enclosed in quotes (' ', " ", or ``).
Types:
Single and double quotes: Use for fixed strings.
Template literals (``): Use for dynamic content and multi-line strings.
Key Features of 0: Insert variables or expressions directly inside the string using ${}.
let firstName = "Ajay";
let age = 21;
console.log(`My name is ${firstName} and I am ${age} years old.`);

For loops >
for (initialization; condition; increment/decrement) {
    // Code to execute
}
for (let i = 0; i < 5; i++) {
    console.log("Iteration:", i);
}
------------
The for...of loop is used to iterate over iterable objects like arrays, strings, Maps, or Sets.
for (let variable of iterable) {
    // Code to execute
}
const fruits = ["Apple", "Banana", "Cherry"];
for (const fruit of fruits) {
    console.log(fruit);
}
-------------
The for...in loop is used to iterate over enumerable properties of an object.
for (let key in object) {
    // Code to execute
}
const person = { name: "John", age: 25, city: "New York" };
for (const key in person) {
    console.log(`${key}: ${person[key]}`);
}
